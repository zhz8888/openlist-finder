# v0.2.0 版本更新

**发布类型**: 功能更新

## 版本亮点

我们很高兴宣布 v0.2.0 版本发布！本次更新带来了 MCP 服务架构升级、文件预览系统重构、Meilisearch 自动同步等多项重磅功能。

## ✨ 新增功能

### MCP 服务 HTTP 传输
将 MCP 服务从 stdio 传输改为 HTTP 传输，提升了服务响应速度和稳定性，方便 AI 工具集成调用。

### 文件预览注册系统
建立统一文件预览注册系统，支持多种文件类型：
- **图片预览**: jpg、jpeg、png、gif、bmp、svg、webp、ico、avif
- **视频预览**: mp4、avi、mkv、mov、wmv、flv、webm
- **音频预览**: mp3、wav、flac、aac、ogg、wma
- **压缩包预览**: zip、rar、7z、tar、gz、bz2
- **文档预览**: PDF、CSV
- **文本预览**: txt、json、xml、yaml、md 等

### Meilisearch 自动同步
实现 Meilisearch 索引自动同步功能，支持手动和定时同步两种模式，让文件检索更加便捷。

### 直连模式提示
当 Meilisearch 服务不可用时，界面会显示直连模式提示，避免用户困惑。

### CI/CD 工作流优化
- 优化 CI 构建流程
- 修复 Windows 构建兼容性问题
- 添加 RPM 包构建支持
- 完善构建产物收集与上传

## 🔨 改进优化

- **Rust 代码格式化**: 统一代码格式，修复 clippy warning
- **性能优化**: 日志组件使用 VecDeque 提升效率、组件 memo 化、useMemo 优化等
- **错误处理增强**: AppInitializer 错误反馈、LogViewerPage 竞态条件修复、Toast 内存泄漏修复
- **UI 改进**: FileRow 和 SearchResultRow 组件提取、文件排序逻辑共享、模态框重构
- **密钥管理优化**: Token 过期自动刷新处理、错误信息改进

## 🐛 问题修复

- 修复文件预览组件懒加载报错
- 修复 Windows 构建时 Generate build summary 步骤的语法错误
- 修复 MCP 服务器 Token 传递问题
- 修复 Toast 通知内存泄漏风险
- 修复 LogViewerPage 加载状态竞态条件
- 移除无法保存的编辑功能
- 移除未使用的 Suspense 导入

## ⚠️ 注意事项

- CI/CD 工作流已在多个平台验证通过（macOS、Windows、Linux）
- Linux 版本现在支持 DEB、RPM 和 AppImage 三种安装包格式

## 升级指南

此版本为功能更新，无需特殊迁移步骤。请确保：
- Node.js >= 24
- Rust >= 1.90
- Tauri CLI 2.x
