# v0.1.0 版本发布

**发布类型**: 首次发布

## 版本亮点

OpenList Finder 首个正式版本发布！这是一款基于 Tauri 2 构建的跨平台文件管理工具，集成 OpenList 服务与 Meilisearch 搜索引擎，支持 MCP 协议供 AI 工具调用。

## ✨ 核心功能

### OpenList 集成
连接 OpenList 服务器，浏览和管理远程文件系统：
- 文件列表浏览
- 文件重命名、删除、复制、移动
- 多服务器管理
- Token 认证

### Meilisearch 搜索
将文件元数据同步至 Meilisearch 索引，实现快速全文检索：
- 索引状态检查
- 搜索模式切换
- 手动和定时同步

### MCP 服务
通过 stdio 暴露文件操作与搜索能力，供 AI 工具集成调用：
- 文件列表查询
- 文件信息获取
- 文件重命名、删除、复制、移动
- 文件编辑
- 搜索与索引同步

### 文件预览与编辑
在线预览与编辑文本文件，支持图片预览。

### 安全存储
使用系统钥匙串安全存储敏感信息（Token、API Key）。

### 主题系统
支持 GitHub Light / GitHub Dark / 跟随系统三种主题模式，基于 Tailwind CSS 自定义实现。

## 🎨 用户界面

- **侧边栏**: 服务器选择、主题切换
- **文件列表**: 排序、搜索、操作菜单
- **设置页面**: 服务器配置、Meilisearch 配置、MCP 配置
- **日志查看器**: 实时日志查看、按等级筛选、分页加载

## 🔧 技术架构

- **桌面框架**: Tauri 2
- **前端**: React 19 + TypeScript
- **构建工具**: Bun
- **UI**: Tailwind CSS 4
- **状态管理**: Zustand
- **路由**: React Router DOM
- **后端**: Rust (reqwest、serde、tokio、chrono、keyring)
